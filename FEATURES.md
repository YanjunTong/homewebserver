# 🎬 HomeMedia Hub - 完整功能说明

一个专业的本地媒体管理系统，提供媒体扫描、缓存、流传输和多页面展示。

---

## 📋 核心功能列表

### 1️⃣ **媒体扫描系统** 📁

#### 自动扫描和入库
- ✅ 支持扫描指定文件夹中的所有媒体文件
- ✅ 自动识别文件类型（图片/视频）
- ✅ 支持的图片格式：`.jpg, .jpeg, .png, .gif, .bmp, .webp, .tiff`
- ✅ 支持的视频格式：`.mp4, .mov, .avi, .mkv, .flv, .wmv`
- ✅ 智能去重：避免重复扫描（已入库文件跳过）
- ✅ 实时反馈：显示添加数、跳过数、失败数

#### 元数据自动提取
- ✅ **图片**：宽度、高度、文件大小、创建时间
- ✅ **视频**：时长、分辨率、文件大小、创建时间
- ✅ 异步非阻塞处理（使用 `asyncio.to_thread()`）
- ✅ 自动处理提取失败情况

#### 分类扫描功能
- ✅ 全部扫描：扫描所有媒体文件夹
- ✅ 图片扫描：仅扫描图片文件夹
- ✅ 视频扫描：仅扫描视频文件夹
- ✅ 自定义路径扫描：支持任意路径

---

### 2️⃣ **缩略图生成系统** 🎨

#### 图片缩略图
- ✅ 使用 Pillow 库自动生成
- ✅ 等比缩放至 300px 高度
- ✅ WebP 格式存储（质量 80）
- ✅ 本地文件系统缓存
- ✅ 缓存检查（已生成的不重复生成）
- ✅ 支持所有常见图片格式

#### 视频缩略图
- ✅ 使用 FFmpeg 提取视频帧
- ✅ 从视频第 5 秒位置提取关键帧
- ✅ 等比缩放至 300px 高度
- ✅ WebP 格式存储
- ✅ 生成失败时显示优雅的占位图
- ✅ 使用 subprocess 执行 ffmpeg 命令
- ✅ 异步非阻塞处理

#### 缓存管理
- ✅ 本地文件系统缓存（/static/thumbnails/）
- ✅ HTTP 缓存控制头（24 小时）
- ✅ 浏览器缓存支持
- ✅ ETag 为缓存验证

### 3️⃣ **API 接口系统** 🔌

#### 媒体查询接口
```
GET /media                      # 分页获取媒体列表
GET /media/count                # 获取媒体总数
GET /media/{id}                 # 获取单个媒体详情
```

#### 缩略图接口
```
GET /media/{id}/thumbnail       # 获取媒体缩略图（WebP 格式）
                                # 返回 FileResponse
                                # 支持 HTTP 缓存
```

#### 媒体流播放接口
```
GET /media/{id}/stream          # 获取媒体文件流
                                # 支持 HTTP Range 请求
                                # 支持断点续传
                                # 根据格式自动识别 MIME 类型
```

#### 媒体扫描接口
```
POST /scan                      # 扫描所有媒体
POST /scan/images               # 扫描图片
POST /scan/videos               # 扫描视频
```

#### API 特性
- ✅ JSON 格式返回
- ✅ 分页支持（skip / limit）
- ✅ 完整的错误处理和状态码
- ✅ 统一响应数据结构
- ✅ Pydantic v2 数据验证

## 🚀 使用指南

### 启动应用
```bash
cd /home/yanjun/workspace/project
uvicorn main:app --host 0.0.0.0 --port 8000
```

### 访问应用
- **首页导航**: http://localhost:8000/static/index.html
- **图片库**: http://localhost:8000/static/images.html
- **视频库**: http://localhost:8000/static/videos.html

```bash
# 扫描所有
curl -X POST http://localhost:8000/scan

# 扫描图片
curl -X POST http://localhost:8000/scan/images

# 扫描视频
curl -X POST http://localhost:8000/scan/videos
```
